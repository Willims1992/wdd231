SkillSwap - Website Project Plan

Site Name

SkillSwap - The name reflects the site's core idea of exchanging skills and micro-jobs among users.

Site Purpose

The SkillSwap platform allows users to post and browse skill-sharing requests, creating an interactive peer-learning community.

Scenarios

  • How can I find someone to teach me a new skill?
  • Where can I post a request for a quick task assistance?

Color Schema

Primary Colors:

  • #007bff (Blue) - Used for headings and action buttons.
  • #f4f4f4 (Light Gray) - Background color for a clean look.

Typography

Fonts Used:

  • Arial - Used for body text.
  • Roboto - Used for headings for a modern, professional look.

Wireframe Sketch

Wireframe layouts will be designed for both **mobile** and **desktop** views, emphasizing user-friendly navigation and accessibility.
